# MeteoBar Roadmap

Planned features and improvements for MeteoBar.

## Planned Features

### Location picker / CoreLocation integration
- Allow users to search for and select locations
- Integrate with macOS CoreLocation for automatic location detection
- Store favorite locations

### User preferences UI
- Settings panel accessible from the forecast window
- Configurable temperature units (Celsius/Fahrenheit)
- Configurable wind speed units (km/h, m/s, mph)
- Adjustable refresh interval
- Option to show/hide specific weather metrics

### Keyboard shortcuts
- Global hotkey (Cmd+Shift+W) to show/hide the forecast panel
- Keyboard navigation within the forecast view

### Multiple locations support
- Add and manage multiple saved locations
- Quick switching between locations from the menu bar
- View forecasts for different locations

### Weather alerts
- Display severe weather warnings
- Notifications for significant weather changes
- Visual indicators for active alerts

### Tests
- Unit tests for weather data processing
- Integration tests for API calls
- UI component tests
